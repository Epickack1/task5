MyShell - Простой командный интерпретатор на C
Описание

MyShell - это простой командный интерпретатор (shell), написанный на языке C. Он поддерживает базовые возможности выполнения команд, включая фоновые процессы, конвейеры (pipes) и перенаправление вывода.
Возможности

    Выполнение внешних команд (ls, cat, grep и т.д.)

    Встроенная команда cd для смены директории

    Поддержка конвейеров (|) для связи команд

    Фоновое выполнение команд (с помощью &)

    Перенаправление вывода в файл (>)

    Обработка завершения дочерних процессов

    Поддержка кавычек для аргументов с пробелами

    Цветное приглашение командной строки

Требования

    Компилятор C (GCC или Clang)

    Система Linux/Unix

Сборка

Для сборки проекта выполните:
bash

gcc -o myshell main.c myshell.c

Или используйте Makefile (если он есть в проекте).
Использование

Запустите скомпилированную программу:
bash

./myshell

Вы увидите приглашение командной строки фиолетового цвета: Myshell $
Примеры команд

    Простая команда:
    text

Myshell $ ls -la

Смена директории:
text

Myshell $ cd /home/user

Конвейер (pipe):
text

Myshell $ ls -l | grep .txt

Перенаправление вывода:
text

Myshell $ ls > output.txt

Фоновое выполнение:
text

Myshell $ sleep 10 &

Команда в кавычках:
text

Myshell $ echo "Hello World"

Структура файлов

    main.c - Основной цикл программы, обработка ввода пользователя

    myshell.h - Заголовочный файл с объявлениями структур и функций

    myshell.c - Реализация функций shell:

        Разбор строки на токены

        Управление памятью

        Выполнение команд

        Обработка сигналов

Ключевые функции

    read_line() - Чтение строки ввода пользователя

    split_line() - Разбор строки на токены с учетом специальных символов

    execute_command() - Выполнение одиночной команды

    child_exec() - Исполнение команды в дочернем процессе

    sigchld_handler() - Обработчик сигнала завершения дочерних процессов
